# 1. 모니터 닫아도 안꺼지게 설정/해제

- 뚜껑 닫아도 안꺼지게 설정
```
sudo pmset -c disablesleep 1
```

- 뚜껑 닫아도 안꺼지게 해제
```
sudo pmset -c disablesleep 0
```
